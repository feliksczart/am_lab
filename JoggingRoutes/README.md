# JoggingRoutes
Aplikacja mobilna napisana w kotlinie oparta na fragmentach i poprawnie wyświetlana na różnych urządzeniach (smartphone, tablet). Służy do obrazowania tras biegowych, rysując ścieżkę na mapie google.
